# Convenções de Nomenclatura

## Camel Case -- Mais Famosa
- Primeira palavra em minúscula
- Palavras seguintes começam com maiúscula
- Exemplo: `let idadeUsuario = 0`

---

## Pascal Case
- Todas as palavras começam com letra maiúscula. Usado para classes, construtores e componentes
- Exemplo: `UsuarioSistema`

---

## Snake Case
- Palavras separadas por underscore (sublinhado)
- Tudo minúsculo
- Muito comum em Python, banco de dados e APIs
- Exemplo: ` nome_completo_casado`

---

## Kebab Case
- Palavras separadas por hífen
- NÃO FUNCIONA COMO NOME DE VARIÁVEL EM JAVASCRIPT
- Exemplo: `menu-principal`
